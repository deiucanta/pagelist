# Pagelist

## Install

```
npm install pagelist
```

```
yarn add pagelist
```

## Usage

```js
import { pagelist } from 'pagelist';

const current = 1;
const total = 500;
const display = 9;

pagelist(current, total, display);
// => 1, 2, 3, 4, 5, 6, 7, _, 500
```
